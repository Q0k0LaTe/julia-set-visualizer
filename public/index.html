About Julia Sets

The Julia set is a fractal that arises from the study of complex dynamics. It's defined as the set of points z in the complex plane for which the function f(z) = zÂ² + c does not escape to infinity when iterated repeatedly.

Different values of the complex constant c produce different Julia sets. Use the sliders to adjust the real and imaginary parts of c, as well as the maximum number of iterations and zoom level.
